# 🔐 Password Generator


A simple and powerful password generator built using HTML, CSS, and JavaScript.


## Features
- Generate strong passwords
- Custom length
- Uppercase, numbers, symbols toggle
- Copy to clipboard


## Tech Stack
- HTML
- CSS
- JavaScript


## How to Run
1. Download the project
2. Open `index.html` in browser
